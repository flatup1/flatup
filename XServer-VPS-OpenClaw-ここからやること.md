# XServer VPS（OpenClaw イメージ）— ここからやること

VPS は「OpenClaw」アプリイメージで稼働中。このあとやることを順番に書きます。

---

## ステップ1：VPS管理画面を開く

1. XServer VPS パネルで、サーバー名の下にある **「VPS管理」** の青いボタンをクリック
2. 開いた画面の内容を確認する（次のステップで使う情報がここにある）

---

## ステップ2：接続情報を確認する

VPS管理画面で、次の3つを探してメモする：

| 確認すること | どこに書いてあるか（例） |
|-------------|--------------------------|
| **IPアドレス** | 「サーバー情報」「接続情報」「IP」など |
| **SSH ユーザー名** | 「SSH アカウント」「ログインユーザー」など |
| **SSH パスワード or 鍵** | 「パスワード」「秘密鍵」など |

- **ウェブコンソール**や**ブラウザコンソール**のボタンがあれば、そこからブラウザで VPS の黒い画面に入れる場合があります
- **OpenClaw の初期パスワード・トークン**や**ダッシュボード URL** の案内があれば、それもメモする

---

## ステップ3：VPS に接続する（どちらか一方）

### 方法A：ウェブコンソールがある場合
- VPS管理画面の **「コンソール」「ウェブコンソール」** などをクリック
- ブラウザに黒い画面（ターミナル）が開いたら、そこで次のステップ4以降のコマンドを実行する

### 方法B：Mac のターミナルから SSH で入る場合
Mac のターミナルで（**実際のユーザー名・IP に置き換え**）：

```bash
ssh ユーザー名@IPアドレス
```

例：
```bash
ssh ubuntu@162.43.90.71
```

パスワードを聞かれたら、ステップ2でメモしたパスワードを入力する。

---

## ステップ4：OpenClaw が動いているか確認する

VPS のコンソール（ウェブ or SSH）で：

```bash
openclaw --version
```

- バージョンが表示されれば、OpenClaw は入っている
- 「コマンドが見つからない」と出る場合は、先に OpenClaw をインストールする必要がある（後述の「OpenClaw が入っていない場合」へ）

---

## ステップ5：OpenClaw を起動する

### すでに起動している場合
- VPS管理画面に **「OpenClaw ダッシュボード」「アクセスURL」** のようなリンクがあれば、それをブラウザで開く
- ログイン用のトークン・パスワードの案内があれば、それを使う

### 自分で起動する場合
コンソールで、**トークン（パスワード）を決めて**実行：

```bash
openclaw gateway --allow-unconfigured --token あなたが決めたパスワード
```

例：
```bash
openclaw gateway --allow-unconfigured --token MyOpenClaw2026
```

- `Gateway failed to start` が出ず、`listening` や `ws://` のような表示が出れば起動成功
- **このトークンは後でブラウザからログインするときに使うので、メモしておく**

---

## ステップ6：パケットフィルターでポートを開ける

ブラウザから VPS の OpenClaw にアクセスするために必要。

1. XServer VPS パネル（最初のメニュー画面）に戻る
2. 左メニューから **「パケットフィルター設定」** を開く
3. **「パケットフィルター設定を追加する」** で以下を追加：
   - **ポート番号:** `18789`
   - **プロトコル:** `TCP`
   - **コメント:** `OpenClaw`
4. 保存する

---

## ステップ7：ブラウザから OpenClaw にログインする

1. Mac のブラウザで、次の URL を開く（**IP は自分の VPS の IP に置き換え**）：
   ```
   http://VPSのIPアドレス:18789
   ```
   例：`http://162.43.90.71:18789`

2. ログイン画面が出たら、**ステップ5で決めたトークン**（または VPS 管理画面に書いてあった初期パスワード）を入力

3. ダッシュボードが表示されれば、ここまでの設定は完了

---

## ステップ8：OpenRouter と Telegram を設定する

ダッシュボードが開いたら：

1. **OpenRouter**
   - 設定画面で「API キー」「プロバイダー」などを開き、OpenRouter の API キー（`sk-or-v1-...`）を入力して保存

2. **Telegram**
   - BotFather で取得したボットトークンと、許可するユーザー（allowFrom）を設定画面で入力して保存

または、VPS のコンソールで：

```bash
openclaw onboard
```

を実行し、画面の指示に従って OpenRouter・Telegram を設定してもよい。

---

## 困ったときの確認

| 状況 | やること |
|------|----------|
| VPS管理の場所がわからない | サーバー名の下の **「VPS管理」** ボタンを押す |
| SSH のユーザー名がわからない | VPS管理画面の「SSH」「接続情報」を確認する。「root」や「ubuntu」のこともある |
| ブラウザで 18789 に繋がらない | ステップ6のパケットフィルターで 18789/TCP を開放したか確認する |
| OpenClaw が入っていない | 下の「OpenClaw が入っていない場合」を実行する |

---

## OpenClaw が入っていない場合（コマンドがないとき）

VPS のコンソールで、順に実行：

```bash
# Node が v22 以上か確認
node -v

# 足りなければ Node を入れてから
sudo npm install -g openclaw@latest

# もう一度確認
openclaw --version
```

その後、**ステップ5** からやり直す。
