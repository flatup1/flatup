# JIN専用：Master Brain & LINE AI 運用マニュアル

このマニュアルは、**「難しいことは抜きで、スマホで操作したい」**というJINさんのために、日々の運用を3つのステップに凝縮したものです。

---

## 1. 【知識の移植】Obsidian に書くだけ（週1回〜）

JINさんの「脳」をAIに同期させる唯一の作業です。

1.  **Obsidian にメモを書く**: 
    *   現場で気づいた「入会率が上がる一言」や「新しいジムの哲学」を、Obsidian の特定のフォルダ（例：`Master_Brain/`）に Markdown で保存します。
2.  **自動同期**: 
    *   Obsidian Git プラグイン（または GitHub への Push）を設定しておけば、書いた瞬間にクラウドへ飛びます。
    *   **AI（Dify/AnythingLLM）**は、この GitHub を見ているので、**「JINさんが書く ＝ AIが賢くなる」**が自動で完結します。

---

## 2. 【LINE接客】通知が来たら「成約」を確認（毎日）

AIが24時間、成田のジムの「受付嬢」として働きます。

1.  **AIが自動返信**: 
    *   深夜でも早朝でも、LINE に問い合わせが来たら AI が「哲学」に基づいた承認と肯定を行い、体験予約へ誘導します。
2.  **JINさんは「予約通知」を待つだけ**: 
    *   予約が入ると、LINE 公式アカウントの管理画面（またはメール）に通知が来ます。
3.  **例外対応（たまに）**: 
    *   AI で解決できない複雑な悩み（例：怪我の相談など）は、LINE 公式アプリから JIN さんが直接返信します。AI が「ここからはオーナーに代わります」と繋ぐ設定も可能です。

---

## 3. 【外脳の操作】Claude Code に「一言」投げる（随時）

スマホの Blink Shell（ターミナル）から、Claude Code を使ってシステムを微調整します。

1.  **Blink Shell を開く**: 
    *   `~/exbrain` で Claude Code を起動。
2.  **一言だけ命令する**: 
    *   「LINE の返信をもっと明るくして」
    *   「Obsidian の新しいフォルダをナレッジに追加して」
3.  **AIが実行**: 
    *   Claude がコードを書き換え、デプロイ（反映）まで勝手に終わらせます。JIN さんは **「A/B/C のどれにする？」** という問いに答えるだけです。

---

## 💡 運用のコツ：トークンとコストを抑えるために

*   **定型文を愛する**: 
    *   「料金」「場所」などの質問は AI に考えさせず、テンプレート（固定回答）で返します。これで API 料金は 0円 に近づきます。
*   **Gemini 1.5 Flash を使う**: 
    *   Dify の設定で Gemini 1.5 Flash（無料枠が広い）を指定すれば、月々のランニングコストは VPS 代（月数千円）だけに抑えられます。

---

## 結論：JINさんがやることは「書くこと」と「決めること」だけ

*   **書く**: Obsidian に自分の哲学を書き溜める。
*   **決める**: AI の提案（A/B/C）から、自分の直感に合うものを選ぶ。

これだけで、成田のジムは JIN さんが寝ている間も「知性と品格」を語り、新しい仲間（会員）を増やし続けます。
