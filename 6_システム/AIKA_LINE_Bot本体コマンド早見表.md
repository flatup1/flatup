# AIKA LINE Bot本体コマンド早見表

これはMacのターミナルではなく、LINEのトーク画面にそのまま送るコマンドです。

## まず覚える

```txt
/help
```

LINE上でコマンド一覧を表示します。忘れたらこれだけ送ればOKです。

```txt
/status
```

AIKAの稼働状態、今日の休業モード、登録中の休業日を確認します。

```txt
/メモ 明日スポンサー文面を確認
```

inbox.md にメモを保存します。分類や実行はせず、夜に整理します。

```txt
/朝
```

board.md を読み、今日死守する3つ、売上を作る3つ、アイデア保管、AIKA提案を表示します。

```txt
/夜
```

inbox.md と board.md を読み、保存前プレビューだけを表示します。この時点ではファイル更新しません。

```txt
/保存
```

直前の `/夜` プレビューを保存します。`/夜` なしでは保存しません。

```txt
/再開
```

done.md を読み、前回止まった作業と次の一手を表示します。

## 動画指示書コマンド

```txt
/動画メモ キッズクラス紹介の縦動画を作りたい
```

Video Factoryの動画アイデアとして保存します。実装やレンダリングはしません。

```txt
/動画案
```

保存した動画メモから、次に作る1本をAIKAが1つだけ提案します。

```txt
/動画指示 怖くない格闘技ジムを伝える縦動画
```

MacのCodex / Claude Codeへ渡すRemotion動画指示書を保存します。

```txt
/動画再開
```

最新の動画指示書に戻り、Macで再開する一手を表示します。

## 競合調査・広告案コマンド

```txt
/リサーチヘルプ
```

競合調査AIの使い方を表示します。

```txt
/競合URL 競合ジム名 | https://example.com | 成田市 | キックボクシング
```

競合ジムの公開URLを登録します。ログインが必要なページや内部APIは登録しません。

```txt
/競合一覧
```

登録した競合URLを確認します。

```txt
/リサーチ
```

登録済みの競合URLを読み、競合調査・広告案・LINE文・Instagramリール台本のレポートをObsidianに保存します。

```txt
/リサーチ最新
```

最新レポートの「今週の結論」だけをLINEで確認します。

## 管理コマンド

```txt
/reset
```

AIモードと会話履歴をリセットします。担当者対応後にAIKAへ戻したいときに使います。

```txt
/closed today
```

今日を休業日にします。その日はAI応答より先に休業メッセージを返します。

```txt
/closed 2026-05-20
```

指定日を休業日にします。日付は `YYYY-MM-DD` 形式です。

```txt
/closed list
```

今設定されている休業日一覧を見ます。

```txt
/open today
```

今日を営業日に戻します。

```txt
/open 2026-05-20
```

指定日を営業日に戻します。

```txt
/closed off
```

登録済みの休業日をすべて解除します。

## 使い方の流れ

1. LINEでAIKAのトークを開く
2. `/help` を送る
3. 必要なコマンドをコピペして送る
4. AIKAの返答を確認する

## 注意

- 重要判断はAIKAだけで完結させない。
- 休業・営業の切り替えは送った直後から反映される。
- `/reset` は会話履歴を消してAIモードへ戻すためのコマンド。
