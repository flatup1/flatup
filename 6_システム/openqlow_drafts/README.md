# OPENQLOW Drafts

OPENQLOW / HelMES 側で生成したSNS下書きのVault控えを置く。

## 扱い

- ここは公開済み投稿ではなく、下書き・承認ログの控え。
- X、Instagram、Threadsなど媒体別に分ける。
- 投稿・予約・送信はJIN承認後のみ。
- Typefullyや各SNSへの反映は、別途人間確認を通す。

## フォルダ

| フォルダ | 内容 |
|---|---|
| `x/` | X向け下書き |
| `instagram/` | Instagram向け下書き |
| `threads/` | Threads向け下書き |

## 保存ルール

- `approval_id` を残す。
- `publication_level` を残す。
- 生成日時を残す。
- 個人情報、APIキー、未承認の顧客対応文は保存しない。
