# FLATUP AIKA OS 中学生にもわかる使い方

## これは何？

AIKAは、FLATUP GYMの仕事を手伝う「LINEで動く秘書」です。

できることは大きく4つです。

1. 忘れないようにメモする
2. 今日やることを整理する
3. 動画や広告の指示書を作る
4. 競合調査やLINE配信文の下書きを作る

大事なこと：
AIKAは勝手に送信しません。
AIKAは勝手に予約確定しません。
AIKAは勝手にお金や会員情報を変更しません。

AIKAは「下書き」と「整理」までです。
最後に決めるのはJINです。

## 全体の箱

AIKA OSは3つの箱で考えます。

```text
INBOX  = とりあえず入れる箱
BOARD  = 今日やることを見る箱
VAULT  = Obsidianに残る記憶の箱
```

迷ったら、まずLINEで `/メモ` します。

## LINEで使う基本コマンド

### 1. メモする

```text
/メモ 明日スポンサー文面を確認する
```

AIKAが `inbox.md` に保存します。
きれいに整理しなくて大丈夫です。

### 2. 朝に見る

```text
/朝
```

今日やることを表示します。

- 今日やらないと危ないこと
- 売上につながること
- アイデア保管
- AIKAからの提案1つ

### 3. 夜に整理する

```text
/夜
```

AIKAが今日の整理案を出します。
この時点では保存しません。

保存してよければ次を送ります。

```text
/保存
```

### 4. 前回の続きに戻る

```text
/再開
```

前回どこで止まったか、次に何をすればいいかを表示します。

## 動画を作りたいとき

### アイデアだけ保存

```text
/動画メモ キッズクラスの楽しさを伝える動画を作りたい
```

### 次に作る動画を1つ提案

```text
/動画案
```

### Codex用の指示書を作る

```text
/動画指示 怖くない格闘技ジムを伝える縦動画
```

これは動画そのものを作るのではなく、MacのCodexやClaude Codeに渡す指示書を作ります。

### 最新の動画指示書に戻る

```text
/動画再開
```

## 競合調査・広告案を作りたいとき

### 使い方を見る

```text
/リサーチヘルプ
```

### 競合URLを登録する

```text
/競合URL 競合ジム名 | https://example.com | 成田市 | キックボクシング
```

`|` で区切ります。

順番はこれです。

```text
名前 | URL | 地域 | 種別
```

### 登録した競合を見る

```text
/競合一覧
```

### レポートを作る

```text
/リサーチ
```

AIKAがObsidianにレポートを保存します。

保存先：

```text
6_システム/flatup-research-ai/reports/YYYY-MM-DD_weekly_research.md
```

レポートにはこれが入ります。

- 競合調査まとめ
- 競合の強み
- 競合の弱み
- FLATUPの勝ち筋
- LINE配信用文章
- Instagramリール台本
- 広告コピー
- 今週やるべきこと

### 最新レポートを見る

```text
/リサーチ最新
```

LINEでは短く「今週の結論」だけ見ます。
全文はObsidianで見ます。

## 管理コマンド

### コマンド一覧

```text
/help
```

忘れたらこれだけ送ればOKです。

### 状態確認

```text
/status
```

AIKAが動いているか、休業日設定があるかを確認します。

### AIモードに戻す

```text
/reset
```

担当者対応後、AIKAをまた動かしたいときに使います。

### 休業日にする

```text
/closed today
```

今日を休業日にします。

```text
/closed 2026-05-20
```

指定日を休業日にします。

### 営業日に戻す

```text
/open today
```

今日を営業日に戻します。

```text
/closed off
```

休業日設定を全部解除します。

## 使い方のおすすめ順

毎朝：

```text
/朝
```

思いついたら：

```text
/メモ 内容
```

広告やSNSを考える日：

```text
/競合一覧
/リサーチ
/リサーチ最新
```

夜：

```text
/夜
/保存
```

次の日：

```text
/再開
```

## やってはいけないこと

- AIKAのLINE文を確認せずそのまま配信しない
- 予約確定をAIKAだけで決めない
- 請求や料金変更をAIKAだけで決めない
- 会員情報をAIKAにそのまま入れない
- 他社サイトのログイン情報を入れない
- CAPTCHA回避や内部API解析をしない

## 一言まとめ

AIKAは、JINの頭の中をLINEに出して、Obsidianに残すための秘書です。

投げ込む。
整理する。
朝に見る。
夜に戻す。
次の日に再開する。

この流れをLINEだけで回せるようにしています。
