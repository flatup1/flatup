# FLATUP ジムコンテンツマーケティング プロンプト集

## 📋 概要

このドキュメントは、格闘技ジム「FLATUP」のコンテンツマーケティング戦略に基づいた、10個の厳選されたプロンプトテンプレート集です。

### 戦略の3層構造

**レイヤー1：入口（超やさしい）**
- ルールは出さない
- 優しさ100%
- 不安の除去だけ

**レイヤー2：中腹（選択）**
- 理念を"物語"で提示
- 「合う合わないがあります」

**レイヤー3：核心（覚悟）**
- ルール提示
- 同意
- 自己選択

> **重要原則**: 最初から核心を見せない

## 🎯 10個のコアプロンプト

### ① テーマ：「優しさは、設計できる」

```
あなたはエモーショナル・アーキテクト。
格闘技ジムFLATUPが「なぜ優しいのか」を、

しかし軽くも見せず、
一本の短い物語として描写してください。

主人公は"気が弱い初心者"。
敵は存在しない。
守られている空気そのものが主役。
```

**活用シーン**: ブランドストーリー、入口コンテンツ、初回接触時のメッセージ
