# MARKETING

SNS、LINE広告、配信、動画制作など販促資料を置く。

生データや古い素材は `5_アーカイブ/` に退避し、ここには参照しやすい資料を残す。

## 正本資料

| 領域 | ファイル/フォルダ | 使い道 |
|---|---|---|
| X / Typefully | `SNS/Typefully_X運用メモ_FLATUP.md` | X投稿の下書き・予約運用 |
| SNS | `SNS/FLATUP_CM_Prompt 2.md` | CMや投稿案のプロンプト |
| SNS | `SNS/FLATUP GYM ブランド構築・SNS戦略 対話履歴.md` | ブランド発信の判断材料 |
| LINE広告 | `LINE広告/` | 広告データ取得・配信報告 |
| LINE配信 | `配信/` | 一斉配信テンプレートと手順 |
| 動画 | `video_factory/` | ショート動画の量産設計 |
| SORA | `動画_SORA/` | AI動画生成プロンプト |

## 運用ルール

- 公開、配信、予約投稿はJIN承認後に行う。
- 1日5投稿まで、オリジナル比率70%以上を守る。
- 自動リプライ、自動いいね、自動フォローは禁止。
- 生データ、PDF、画像、動画、txtは基本的にGitHub管理しない。
- 参考資料を使う時は、FLATUPの経験と言葉に置き換える。
