## 運用ルール

- ルート直下には入口ファイルだけ置く。
- FLATUPGYM AI全体の入口は `00_CORE/FLATUPGYM_AI_HOME.md`。
- 現行システム仕様は `6_システム/` に置く。
- AIKAやジム理念の核は `00_CORE/` に置く。
- 顧客対応の型は `01_SKILLS_CUSTOMER/` に置く。
- SNS、広告、配信、動画は `02_SKILLS_MARKETING/` に置く。
- JINの思考整理AIの運用ルールは `03_SKILLS_INTERNAL/jin_copilot_protocol.md` に置く。
- JINの未整理メモ、整理済みタスク、再開メモは `4_日記/JIN_AI_*.md` に置く。
- FLATUPGYM AIの改善・学習ログは `4_日記/FLATUPGYM_AI_学習ログ.md` に置く。
- 古いロードマップ、引き継ぎ、未分類ファイルは `5_アーカイブ/` に置く。
- アーカイブ内の手順は現行手順として実行しない。
