# Vault構造リファレンス（人間用）

## フォルダ一覧

| フォルダ | 用途 |
|---|---|
| `00_CORE/` | ジム基本情報・料金・キャンペーン（Claudeが頻繁に参照） |
| `01_SKILLS_CUSTOMER/` | 顧客対応スキル（LINE返信・予約・入会・クチコミ・クレーム） |
| `02_SKILLS_MARKETING/` | 販促スキル（UIZIN・SNS・POP・ブランド） |
| `03_SKILLS_INTERNAL/` | 内部スキル（AIKAトーン・文章フィルター） |
| `04_日記/` | JINの運営ログ（本番非公開） |
| `05_アーカイブ/` | 終了済キャンペーン等 |
| `06_システム/` | LINE Bot・VPS仕様書 |
| `99_REFERENCE/` | 人間用参考資料（Claudeに毎回読ませない） |

## ファイル追加ルール
- ルート直下に散らかさない
- 必ず上記フォルダに分類
- Skillファイルは150行以内
