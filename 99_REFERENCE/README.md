## GitHub管理

- Markdownの参考資料はGitHub管理してよい。
- `exbrain/` は外部テンプレート由来のため、README以外は基本的に管理外。
- 参考資料を正本に昇格する時は、要約して該当フォルダへ移す。

## 使い方

1. まず正本を見る。
2. 足りない背景がある時だけ、このフォルダを検索する。
3. 参考資料の内容をそのまま実行せず、現行ルールに合わせて要約する。
4. 新しい知見として使う場合は、`4_日記/FLATUPGYM_AI_学習ログ.md` か該当する正本ファイルへ昇格する。
